# Simulation Configurations

This directory is intended for storing simulation configuration files (YAML).

## Usage
- **Presets**: You can save standardized environmental models here for reuse.
- **User Workspaces**: AI Agents and users are encouraged to save their generated configurations in this folder to keep the project root clean.
- **Example**: See `example.yaml` for a template of a complete simulation setup.
